OPTION 3: RENDER CLOUD DEPLOYMENT
1. Connect GitHub repo: github.com:bhola-dev58/JLiveChats.git
2. Deploy Branch: main
3. Build Command: mvn clean package
4. Start Command: java -jar target/jlivechats-1.0.0.jar.original
5. Environment Variables:
   - PORT=8080 (auto-configured by Render)
   - GOOGLE_CLIENT_ID=your_oauth_client_id
   - GOOGLE_CLIENT_SECRET=your_oauth_secret
